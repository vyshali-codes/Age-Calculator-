# Age Calculator

A modern, responsive, and highly polished Age Calculator web application built with React, TypeScript, and Tailwind CSS. This app provides detailed age breakdowns and a countdown to your next birthday with smooth animations.

## 🚀 Features

- **Custom Date Picker**: A bespoke calendar component with manual typing support (DD/MM/YYYY).
- **Detailed Age Breakdown**: Calculate your exact age in years, months, and days.
- **Total Stats**: View your age in total days, hours, and minutes.
- **Next Birthday Countdown**: Real-time countdown to your upcoming birthday.
- **Polished UI/UX**:
  - **Lavender Theme**: A clean, high-contrast minimal aesthetic.
  - **Smooth Animations**: Count-up effects for numbers and staggered entrance animations for results using `motion`.
  - **Responsive Design**: Fully optimized for mobile, tablet, and desktop screens.
- **Robust Validation**: Real-time feedback for invalid formats, future dates, and non-existent dates (e.g., Feb 30th).

## 🛠️ Tech Stack

- **React 19**: Frontend framework.
- **TypeScript**: For type-safe development.
- **Tailwind CSS 4**: Utility-first styling.
- **Luxon**: Powerful date and time manipulation.
- **Motion (Framer Motion)**: Fluid animations and transitions.
- **Lucide React**: Beautiful, consistent iconography.

## 📦 Installation & Setup

1. **Clone the repository** (if applicable).
2. **Install dependencies**:
   ```bash
   npm install
   ```
3. **Run the development server**:
   ```bash
   npm run dev
   ```
4. **Build for production**:
   ```bash
   npm run build
   ```

## 📝 Usage

1. Enter your birth date using the custom input field or select it from the calendar.
2. Click the **"Calculate Age"** button.
3. Explore your detailed age stats and see how long until your next celebration!
